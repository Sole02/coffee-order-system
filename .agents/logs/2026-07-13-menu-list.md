# 커피 메뉴 목록 조회 API

- 날짜: 2026-07-13
- 엔드포인트: `GET /menus`
- 요구사항: Menu 테이블 전체 목록을 (id, name, price)로 응답

## 생성/수정 파일
- `domain/menus/dto/response/MenuResponse.java` (신규)
- `domain/menus/service/MenuService.java` (신규)
- `domain/menus/controller/MenuController.java` (신규)

## 구현 메모
- `domain/user`(`PointController`, `PointChargeResponse`) 스타일을 그대로 따름: Response DTO는 `@Getter`만 + final 필드/생성자.
- 엔티티(`Menu`)를 그대로 응답하지 않고 `MenuResponse`로 변환.
- 조회 전용 메서드라 `MenuService.getMenus()`에 `@Transactional(readOnly = true)` 적용.
- 메뉴가 없어도 예외 없이 빈 리스트 반환 (별도 에러코드 불필요).
